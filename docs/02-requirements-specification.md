# Functional Requirements

## Authentication

FR-001:
Users shall be able to register an account.

FR-002:
Users shall be able to log in.

FR-003:
Users shall have different roles.


## Venue Management

FR-004:
Venue owners shall be able to create venues.

FR-005:
Venue owners shall be able to add courts.


## Booking

FR-006:
Players shall be able to create bookings.

FR-007:
The system shall prevent overlapping bookings.

FR-008:
Players shall be able to cancel bookings.
